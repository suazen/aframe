# aframe框架
**项目快速开发脚手架**

### 项目结构  

```yaml
aframe
--aframe-parent 项目依赖包
--aframe-starter 启动模块
--aframe-framework 框架包
----framework-core 基础支撑
----framework-security 权限控制
--aframe-system 系统模块
----system-core
----system-service
--aframe-auth 鉴权模块
...
```

- hutool
